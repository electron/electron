# FlexLayoutConfig Object

* `orientation` string - one of:
  * `horizontal`
  * `vertical`
* `mainAxisAlignment` string - Main axis alignment type. One of the following:
  * `start`
  * `center`
  * `end`
  * `stretch`
  * `baseline`
* `crossAxisAlignment` string - Cross-axis alignment type. One of the following:
  * `start`
  * `center`
  * `end`
  * `stretch`
  * `baseline`
* `interiorMargin` Object
  * `top` number
  * `bottom` number
  * `left` number
  * `right` number
* `minimumCrossAxisSize` number
* `collapseMargins` boolean
* `includeHostInsetsInLayout` boolean
* `ignoreDefaultMainAxisMargins` boolean
* `flexAllocationOrder` string - one of:
  * `normal`
  * `reverse`
