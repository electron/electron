# SafeStorageEncryptor Object

* `encryptString` Function\<Buffer\> - Encrypts a string using the
  asynchronously acquired encryptor.
  * `plainText` string
* `decryptString` Function\<[DecryptStringAsyncReturnValue](decrypt-string-async-return-value.md)\> - Decrypts a string using the
  asynchronously acquired encryptor.
  * `encrypted` Buffer
